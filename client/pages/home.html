A fast way to get started building MEAN.

Built-in authentication, account management, navigation, and responsive views.

Download and Update

Get the seed project as a .zip file or clone it from the git repo.

The MEAN Seed is easy to update. We use Bower to manage client app dependencies and npm for server and build system dependencies.

Configure

Add your configurations for a few simple options then you're free to add content and styling or dig into the client and server apps and hack away.

Read the configuration docs for detailed instructions.

Develop and Build

The MEAN Seed simplifies local development with a custom batch script that updates dependencies, starts a local mongoDB instance, and launches a node server. Plus, we use Grunt for autoprefixing, SASS compilation, and distribution builds.
